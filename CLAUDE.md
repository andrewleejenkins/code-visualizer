# Code Visualizer

## Project Overview
Next.js 15 app for visually exploring codebases. Analyzes dependencies, components, routes, and models.

## Commands
- `npm run dev` - Start development server (localhost:3000)
- `npm run build` - Production build

## Tech Stack
- Next.js 15 (App Router), React 19, TypeScript, Tailwind CSS 4
- Mermaid.js for diagrams, Zustand for state

## Key Directories
- `app/api/` - Analysis endpoints (analyze, insights, read-file)
- `components/architecture/` - All UI components
- `lib/` - Store, types, analyzer logic
